✨ About The Project

Ramadan 1447 Legacy OS is a fully interactive, beautifully designed Ramadan web dashboard that blends spirituality with modern UI/UX design.

This project was built to help users stay spiritually organized and productive throughout Ramadan using digital tools designed with meaning and intention.

It combines faith, technology, and aesthetic design into one immersive web experience.
